# Ideas de Mejora al Sistema LMB

Este documento registra propuestas de mejora detectadas durante la construcción y uso del template.

## ⚠️ IMPORTANTE

**docs/sistema-lmb.md tiene `editable: false`**

Si un agente detecta una mejora al sistema durante cualquier sesión, debe documentarla aquí usando el formato establecido, NO modificar directamente el documento del sistema.

---

## Formato de Registro

```markdown
## IDEA-[ID]: [Título breve]
- **Detectado por**: [rol] (sesión X)
- **Fecha**: YYYY-MM-DD
- **Descripción**: [qué mejorar y por qué]
- **Propuesta**: [texto específico o cambio sugerido]
- **Sección afectada**: [dónde iría en sistema-lmb.md]
- **Estado**: Pendiente revisión | Aprobada | Rechazada | Implementada
- **Notas**: [comentarios adicionales]
```
