## Reglas de Copywriting Aplicadas

### ✅ SIEMPRE

- [ ] Lenguaje del perfil de audiencia (evitar jerga si B2C)
- [ ] Estructura problema → solución → beneficio
- [ ] CTAs en segunda persona ("Solicita TU consulta")
- [ ] Párrafos cortos (3-4 líneas, escaneables)
- [ ] Listas con bullets (fáciles de leer)
- [ ] Densidad de keywords natural (LSI, semánticas)

- [ ] Copiar textos de competencia
- [ ] Lenguaje genérico sin personalizar
- [ ] CTAs pasivos ("Más información" → "Descarga guía gratuita")
- [ ] Párrafos largos sin espacios
- [ ] Jerga técnica para audiencia B2C
